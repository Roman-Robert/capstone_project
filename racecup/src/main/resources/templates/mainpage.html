Welcome to RaceCup!

The service was developed as part of the final project of the EPAM UpSkillMe program on the Java Software development course.

Main functions:

  1. Race schedule:
    • Create and view future bike races
    • Showing the results of past races
  2. Rating calculation:
    • Personal rating of athletes
    • Team rating
  3. Distribution by roles has been implemented, which expands the range of available functions for registered users:
    • User
    • Athlete
    • Organizer
    • Admin

Stack of applied technologies:

  • Java 11
  • MySQL
  • Spring framework
  • Thymeleaf
  • JUnit
  • HTML

RaceCap is a project developed as part of a curriculum designed to demonstrate Java development skills.
